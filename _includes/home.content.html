关于github pages

Github Pages

简单来说 GitHub Pages 是可以被认为是用户编写的、托管在 GitHub 上的静态网页的技术。

如何开始一个github pages

1. 首先在你的github 帐号内 新建一个名为 YOURNAME.github.io的 库（Repository），YOURNAME请自行替换成你自己的名字

2. 接着回到本地直接使用git clone

 git clone https://github.com/YOURNAME/YOURNAME.github.io
这时候已经可以直接编写页面了,并且 YOURNAME.github.io 已经可以访问

关于jekyll

来自官方的简介介绍: 将纯文本转换为静态博客网站Jekyll官方中文站

关于安装与使用, 推荐是在linux系统上通过gem来安装

 ~ $ gem install jekyll bundler 
 ~ $ jekyll new my-awesome-site
 ~ $ cd my-awesome-site
 ~/my-awesome-site $ bundle install
 ~/my-awesome-site $ bundle exec jekyll s

之后打开浏览器 http://localhost:4000 便可看到jekyll生成的页面
